# Smart Attendance System Using Face Recognition

## Overview
A Flask-based web application that automates attendance tracking using real-time face recognition technology. The system eliminates manual attendance processes and provides accurate, contactless student monitoring.

## Features
- **Face Registration & Recognition** - Register students via facial data and mark attendance automatically
- **Dual Dashboard System** - Separate interfaces for students and faculty
- **Real-time Attendance Tracking** - Live attendance marking with face detection
- **Analytics & Reporting** - Generate attendance reports and analytics
- **User Management** - Secure authentication and role-based access
- **Mobile Responsive** - Accessible on all devices

## Technologies Used
- **Backend:** Flask, Python
- **Face Recognition:** OpenCV, face_recognition library
- **Database:** SQLite/MySQL
- **Frontend:** HTML, CSS, JavaScript, Bootstrap
- **Additional Libraries:** NumPy, Pandas for data processing
